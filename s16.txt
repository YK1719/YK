Q1. Write a Python program using regular expressions to validate whether a given string matches the
pattern of three-letter words such as bit, but, bat, hit, hut, hat.

import re

# Pattern: first letter b or h, second letter i/a/u, third letter t
pattern = r'^[bh][iau]t$'

word = input("Enter a word: ")

if re.match(pattern, word):
    print("Valid word (matches pattern)")
else:
    print("Invalid word")
