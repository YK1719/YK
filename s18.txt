Q1. Write a Python program to list all files and folders in the current directory using os.listdir().

import os

# Get current working directory
current_dir = os.getcwd()

# List all files and folders
items = os.listdir(current_dir)

print("Current Directory:", current_dir)
print("\nFiles and Folders:")

for item in items:
    print(item)

Q2. Write a Python program to validate mobile number based on following criteria
a) The first digit should contain numbers between 6 to 9.
b) The rest 9 digit can contain any number between 0 to 9.
c) The mobile number can have 11 digits also by including 0 at the starting.
d) The mobile number can be of 12 digits also by including 91 at the starting

import re

def validate_mobile(number):
    """
    Validate mobile number based on following rules:
    a) 10 digits starting with 6-9
    b) Can include 0 at start (11 digits)
    c) Can include 91 at start (12 digits)
    """
    pattern = r'^(?:0|91)?[6-9]\d{9}$'
    
    if re.fullmatch(pattern, number):
        return True
    else:
        return False

# Input
mobile = input("Enter mobile number: ")

# Validation
if validate_mobile(mobile):
    print("Valid mobile number")
else:
    print("Invalid mobile number")

Q2. Build a package education with a sub package courses, containing modules science.py and
arts.py. Each module should have a function get_subjects(). Write a python program to display all
subjects.

Science.py
def get_subjects():
    return ["Physics", "Chemistry", "Biology", "Mathematics"]

arts.py
def get_subjects():
    return ["History", "Geography", "Literature", "Psychology"]

main.py
from education.courses import science, arts

# Get subjects
science_subjects = science.get_subjects()
arts_subjects = arts.get_subjects()

print("Science Subjects:")
for subject in science_subjects:
    print("-", subject)

print("\nArts Subjects:")
for subject in arts_subjects:
    print("-", subject)
