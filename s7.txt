
Write a python program to accept a string, extract all words that start with a capital letter and remove all special characters from a string. Display modified string. (Accept input from user)
import re
# Accept input from user
text = input("Enter a string: ")

# Extract words starting with a capital letter
capital_words = re.findall(r'\b[A-Z][a-zA-Z]*\b', text)

# Remove all special characters (keep letters, numbers, and spaces)
modified_string = re.sub(r'[^a-zA-Z0-9\s]', '', text)

# Display results
print("Words starting with capital letters:", capital_words)
print("Modified string:", modified_string)

Create a Web Page using Flask which contains following information about your Library Book List a) Show the library name as a heading. b) Display all available books in a table with title and author. c) Add quote about reading in bold and blue color.

App.py

from flask import Flask, render_template
app = Flask(__name__)
@app.route('/')
def home():
    library_name = "City Library"

    books = [
        {"title": "To Kill a Mockingbird", "author": "Harper Lee"},
        {"title": "1984", "author": "George Orwell"},
        {"title": "Pride and Prejudice", "author": "Jane Austen"},
        {"title": "The Great Gatsby", "author": "F. Scott Fitzgerald"}
    ]
   quote = "A reader lives a thousand lives before he dies."
   return render_template("index.html",
                           library_name=library_name,
                           books=books,
                           quote=quote)
if __name__ == '__main__':
    app.run(debug=True)

Write a python program to accept file name. If file exist copy contents of one file into another file and display contents of new file along with total number of lines, words and characters otherwise give appropriate message.

import os

# Accept file name from user
source_file = input("Enter source file name: ")
destination_file = input("Enter destination file name: ")

# Check if file exists
if os.path.exists(source_file):

    # Open source file and read content
    with open(source_file, 'r') as f:
        content = f.read()

    # Write content to destination file
    with open(destination_file, 'w') as f:
        f.write(content)

    # Display content of new file
    print("\nContents of new file:")
    print(content)

    # Count lines, words, characters
    lines = content.split('\n')
    words = content.split()
    characters = len(content)
    print("\nTotal number of lines:", len(lines))
    print("Total number of words:", len(words))
    print("Total number of characters:", characters)
else:
    print("File does not exist!")
