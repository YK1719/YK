Write a Python program to create a list of elements. Shuffle the elements of a given list and display the list. [Hint: Use random.shuffle()]

import random

# Create a list
lst = [10, 20, 30, 40, 50]

print("Original List:", lst)

# Shuffle the list
random.shuffle(lst)

print("Shuffled List:", lst)
